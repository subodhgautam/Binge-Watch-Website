# Binge Watch Website
Watch movies and shows for free.

## Context for this.
I wanted to learn web develpment in 2024. However, it got delayed to 2026. In 2024, the first thing I wanted to build was a homepage like netflix because I basically had no idea on how to create those accordions and I didn't even know we could use the dev tools to see how they made it.

## About the project
This is a simple homepage inspired from Netflix. In the future I want to learn backend make this a project from which I can watch movies for free. (Without ads and buffering, just smooth playback.)

## Tech stack
- HTML
- TAILWIND CSS
- JAVASCRIPT


## Installation:
1. Clone the repository:
    ```bash
    git clone https://github.com/subodhgautam/Binge-Watch-Website.git
    ```

2. Navigate to `index.html` file to open in your browser to view the App.
